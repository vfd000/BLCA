BLCA Tools

Boulevard Lane Community Association - Digital Tools and Utilities

About

This repository contains various tools and utilities developed for the Boulevard Lane Community Association (BLCA). These tools are designed to help streamline community operations and improve communication.

Available Tools

QR Code Generator

A Python script that generates QR codes from URLs. Perfect for creating quick links to community resources and events.

Features:
  • Generate QR codes from any URL
  • Customizable output location
  • High-quality image output

Getting Started

To use these tools, clone the repository and install the required dependencies:

pip install -r requirements.txt
